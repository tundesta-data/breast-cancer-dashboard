# 📊 Breast Cancer Survival & Treatment Analytics Dashboard

## 📌 Project Overview
This project analyses breast cancer patient data to uncover insights on survival rates, treatment effectiveness, and recurrence patterns using Power BI.

## 🛠️ Tools Used
- Power BI (Data Visualization & DAX)
- SQL (Data Modeling)
