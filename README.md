# breast-cancer-dashboard
Breast Cancer Survival &amp; Treatment Analytics Dashboard (Power BI)
📊 Breast Cancer Survival & Treatment Analytics Dashboard
🧠 Overview

This project analyses breast cancer patient data to uncover insights on survival rates, treatment effectiveness, and recurrence patterns using Power BI.

📁 Dataset
Patient demographics
Cancer stage
Treatment type
Survival & recurrence status
📌 Key KPIs
Total Patients
Survival Rate
Mortality Rate
Recurrence Rate
Average Survival Months
📈 Key Insights
Survival rates decrease significantly in advanced cancer stages
Certain treatment combinations improve survival outcomes
Recurrence patterns vary across age groups and treatment types
🛠 Tools Used
Power BI
SQL
Data Modeling (DAX)
